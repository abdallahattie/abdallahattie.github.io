Welcome — this is the first post on my new site.

I've wanted a small corner of the web for a while: one place that says who I
am, how to reach me, and what I've been thinking about. This is it.

## What to expect here

I work in AI and data science consulting, so most of what I write will orbit
around that:

- **AI & machine learning in practice** — what actually works when you ship
  models to production, beyond the demo.
- **Data science craft** — tooling, workflows, and the unglamorous parts that
  make projects succeed.
- **Notes from consulting** — lessons learned working with different teams,
  codebases, and problems.

Posts will be short and irregular. I'd rather publish one useful idea than
pad a schedule.

## How this site works

For the curious: this site is built with [Astro](https://astro.build), styled
with plain CSS, and deployed to GitHub Pages. Every post is a Markdown file in
a git repository — for example, this post starts like this:

```markdown
---
title: 'Hello, world'
description: 'Why I built this site, and what I plan to write about here.'
pubDate: 2026-06-12
---

Welcome — this is the first post on my new site.
```

If you want to follow along, there's an [RSS feed](/rss.xml). Or just
[email me](mailto:Abdallah@metaseed.ai) — I'm always happy to talk shop.
